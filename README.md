# Snake Game - Projekt Zespołowy WSB

## Opis
Prosta gra Snake napisana w C# działająca w konsoli. Projekt stworzony w ramach zaliczenia przedmiotu.

## Instrukcja Uruchomienia
1. Sklonuj repozytorium.
2. Otwórz plik `.sln` w Visual Studio.
3. Kliknij "Start" lub F5.

## Sterowanie
* Strzałki: Poruszanie wężem.
* Cel: Zbieraj gwiazdki (*), aby zdobywać punkty. Unikaj ścian i własnego ogona.
